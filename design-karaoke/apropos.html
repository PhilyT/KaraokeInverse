ABOUT

Le but de cette aplication est de faire l'inverse de karaoké, ce qui peut signifier que le chanteur n'a pas besoin qu'un orchestre soit présent pour chanter. Dans le karaoké, le texte défile sur l'écran et une personne essaie de le chanter plus ou moins justement.

L'application fait l'inverse, c'est à dire a partir d'un son, à partir d'une note de musique jouée ou un fichier audio, analyse les fréquences et affiche les notes correspondantes sur une partition à l'écran. Le fonctionnement est donc globalement celui d'un accordeur.

L'application utilise des versions instrumentales de chansons au format MP3. Les fichiers servent de base musicale pour les logiciels spécialisés dans ce domaine. Elle propose aussi la création en ligne de playbacks pour instruments permettant de modifier chaque piste individuellement (guitare, batterie, basse...).

CONCEPTS ET USAGES

L'application est réaliser avec les technologies suivants :

  • Capture de l’audio en HTML5 : La montée de l’HTML5 a apporté une transition remarquable en puissance de l’accès au matériel du périphérique. La géolocalisation (GPS), L'API d'orientation et l'API Web Audio (matériel audio), sont des exemples de montée.
  • Mise en forme de l’interface de l’application : CSS & Bootstrap
  • VexFlow : est une API de rendu de notation de musique en ligne open source. Il est entièrement écrit en JavaScript, et fonctionne correctement dans le navigateur. VexFlow prend en charge HTML5 Canvas et SVG.

    • L'application se lance, il faut autoriser l'utilisation de votre micro si vous utilisez le playbacks pour les instruments, et si vous utilisez une versions instrumentales de chansons au format MP3, vous pouvez choisir un fichier en cliquant sur la bouton "Choisissez un fichier", et cliquer sur la bouton play pour lancer la lecture, ou pause pour faire une pause. il y a aussi une bouton effacer qui permet de supprimer les notes apparaître dans la partition. et si vous voulez changer BPM du rythme, il suffit de cliquer sur les fléche de votre clavier, après vous pouvez aussi choisissez la forme de visualisation.
